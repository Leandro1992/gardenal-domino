# Interface do Domin\u00f3 Gardenal - Atualizado

## \ud83d\udcf1 Design Mobile-First

A interface foi desenvolvida com foco em dispositivos m\u00f3veis, garantindo uma experi\u00eancia otimizada em smartphones e tablets, com suporte total para desktop.

## \ud83c\udfa8 Design System

- **Tailwind CSS 3.4**: Framework CSS utility-first para estiliza\u00e7\u00e3o responsiva\n- **Lucide React**: \u00cdcones modernos e consistentes\n- **Cores prim\u00e1rias**: Azul (#0ea5e9) como cor principal\n- **Componentes**: Sistema de componentes reutiliz\u00e1veis em `components/ui/`\n- **Logo**: Logo Gardenal integrada (logo.png, logo-icon.png, favicon.png)\n\n## \ud83d\udcdd P\u00e1ginas Implementadas\n\n### 1. Login (`/login`)\n- Tela de autentica\u00e7\u00e3o com email e senha\n- Valida\u00e7\u00e3o de formul\u00e1rio\n- Feedback de erros\n- Design responsivo e acess\u00edvel\n- Logo Gardenal\n\n### 2. Dashboard (`/`)\n- Bem-vindo personalizado com nome do usu\u00e1rio\n- **6 cards de estat\u00edsticas**:\n  - Vit\u00f3rias (verde, \ud83c\udfc6)\n  - Derrotas (vermelho, \ud83c\udfaf)\n  - Lisas Aplicadas (amarelo, \ud83d\udd25)\n  - Lisas Levadas (roxo, \ud83d\ude41)\n  - Total de Partidas (azul, \ud83c\udfc6)\n  - Partidas em Andamento (azul, \ud83d\udcc8)\n- Lista de partidas ativas com placar\n- Hist\u00f3rico de 5 partidas recentes\n- **Destaque visual correto**: verde para vencedor (quem N\u00c3O chegou a 100), vermelho para perdedor\n- Bot\u00e3o de a\u00e7\u00e3o r\u00e1pida \"Nova Partida\"\n\n### 3. Nova Partida (`/games/new`)\n- Sele\u00e7\u00e3o de 4 jogadores (2 por time)\n- **Campo de busca** para filtrar jogadores por nome/email\n- Interface intuitiva para forma\u00e7\u00e3o de times\n- Valida\u00e7\u00e3o autom\u00e1tica (2 jogadores por time)\n- **Valida\u00e7\u00e3o**: jogador n\u00e3o pode estar em outra partida ativa\n- Feedback visual da sele\u00e7\u00e3o\n- Bot\u00e3o de criar desabilitado at\u00e9 ter 4 jogadores\n\n### 4. Detalhes da Partida (`/games/[id]`)\n- Placar em tempo real com **nomes reais dos jogadores** (n\u00e3o \"Time A/B\")\n- Registro de rodadas (aceita valores vazios como 0)\n- **Valida\u00e7\u00e3o**: apenas membros da partida podem adicionar rodadas\n- Hist\u00f3rico completo de todas as rodadas\n- Indicador de vit\u00f3ria quando finalizada\n- **L\u00f3gica correta**: vencedor = quem N\u00c3O chegou a 100 pontos\n- Badge especial para vit\u00f3ria \"Lisa\"\n- **Anima\u00e7\u00e3o de Lisa**: imagem customizada (ivo.jpeg) com bounce, fade-in, 5 segundos\n- Trof\u00e9u destacando o vencedor\n- Bot\u00e3o voltar para dashboard\n\n### 5. Todas as Partidas (`/games`)\n- Lista completa de todas as partidas do sistema\n- **Filtros de status**: Todas, Em Andamento, Finalizadas\n- **Busca por nome do jogador**\n- **Filtro por data**\n- Bot\u00e3o limpar todos os filtros\n- Cards com informa\u00e7\u00f5es resumidas\n- Indicadores visuais de status\n- **Trof\u00e9u destacando vencedor** (quem N\u00c3O chegou a 100)\n- Badge de Lisa\n- Contador de partidas encontradas\n- Nomes dos jogadores exibidos\n\n### 6. Ranking (`/ranking`) **NOVO!**\n- **Ranking geral** de todos os jogadores\n- Sistema de pontua\u00e7\u00e3o (Score):\n  - Vit\u00f3ria: **+1 ponto**\n  - Lisa aplicada: **+2 pontos**\n  - Derrota: **-1 ponto**\n  - Lisa tomada: **-2 pontos**\n- **Medalhas para top 3**:\n  - 1\u00ba lugar: \ud83c\udfc6 Trof\u00e9u de ouro (fundo amarelo)\n  - 2\u00ba lugar: \ud83e\udd48 Medalha de prata (fundo cinza)\n  - 3\u00ba lugar: \ud83e\udd49 Medalha de bronze (fundo \u00e2mbar)\n- Cores diferenciadas para posi\u00e7\u00f5es de destaque\n- **Destaque visual** para o usu\u00e1rio logado (anel azul)\n- Estat\u00edsticas completas de cada jogador:\n  - Total de partidas\n  - Vit\u00f3rias / Derrotas\n  - Lisas aplicadas / tomadas\n  - Score final (verde se positivo, vermelho se negativo)\n- **Legenda** explicando c\u00e1lculo do score\n- Cards responsivos com estat\u00edsticas detalhadas no mobile\n- Ordena\u00e7\u00e3o: score > vit\u00f3rias > lisas aplicadas\n\n### 7. Gest\u00e3o de Usu\u00e1rios (`/admin/users`) *Apenas Admin*\n- Cria\u00e7\u00e3o de novos usu\u00e1rios\n- Lista de todos os usu\u00e1rios\n- Redefini\u00e7\u00e3o de senha (sem precisar da antiga)\n- Identifica\u00e7\u00e3o visual de tipo de conta (admin/user)\n- Badge de destaque para administradores\n\n### 8. Configura\u00e7\u00f5es (`/settings`) **NOVO!**\n- **Edi\u00e7\u00e3o de perfil**: alterar nome de exibi\u00e7\u00e3o\n- **Altera\u00e7\u00e3o de senha**: requer senha antiga para seguran\u00e7a\n- Dois formul\u00e1rios separados para clareza\n- Valida\u00e7\u00f5es:\n  - Nome m\u00ednimo 3 caracteres\n  - Senha antiga correta\n  - Nova senha m\u00ednimo 6 caracteres\n- Feedback de sucesso/erro com componente Alert\n\n## \ud83c\udfaf Recursos de UX\n\n### Responsividade\n- **Mobile (< 640px)**: Menu hamb\u00farguer, layout vertical, cards empilhados\n- **Tablet (640px - 1024px)**: Layout grid adaptado, 2-3 colunas\n- **Desktop (> 1024px)**: Sidebar fixa, layout horizontal, 6 colunas de stats\n\n### Navega\u00e7\u00e3o\n- **Mobile**: Menu hamb\u00farguer no topo\n- **Desktop**: Sidebar persistente \u00e0 esquerda\n- **5 se\u00e7\u00f5es principais**:\n  1. In\u00edcio (\ud83c\udfe0 Dashboard)\n  2. Partidas (\ud83c\udfc6 Lista e cria\u00e7\u00e3o)\n  3. Ranking (\ud83c\udfc5 Ranking geral)\n  4. Configura\u00e7\u00f5es (\u2699\ufe0f Perfil e senha)\n  5. Usu\u00e1rios (\ud83d\udc65 Apenas admin)\n- **Logo \"Domin\u00f3 Gardenal\"** no header (mobile e desktop)\n- Breadcrumbs visuais com bot\u00f5es de voltar\n- Links contextuais entre p\u00e1ginas relacionadas\n- Indica\u00e7\u00e3o visual de p\u00e1gina ativa no menu\n\n### Feedback Visual\n- **Loading spinners** em todas as opera\u00e7\u00f5es ass\u00edncronas\n- **Alertas** de erro (vermelho) e sucesso (verde)\n- **Badges** de status:\n  - Verde: Em Andamento\n  - Cinza: Finalizada\n  - Amarelo: Lisa\n  - Azul: Voc\u00ea / Admin\n- **Cores sem\u00e2nticas**:\n  - Verde: vit\u00f3ria, sucesso, ativo, vencedor\n  - Vermelho: derrota, erro, perigo, perdedor\n  - Amarelo: lisa aplicada, alerta\n  - Roxo: lisa tomada\n  - Azul: informa\u00e7\u00e3o, prim\u00e1rio, destaque pessoal\n- **Estados hover** em bot\u00f5es, cards e links\n- **Anima\u00e7\u00f5es**:\n  - Fade-in para alertas\n  - Bounce para anima\u00e7\u00e3o de Lisa\n  - Spin para loading\n  - Hover transitions\n\n### Componentes Reutiliz\u00e1veis\n\n#### Button (`components/ui/Button.tsx`)\n```tsx\n<Button variant=\"primary\" size=\"md\" fullWidth disabled={loading}>\n  {loading ? <Loader2 className=\"animate-spin\" /> : 'Texto'}\n</Button>\n```\n- **Variantes**: `primary` (azul), `secondary` (branco/borda), `danger` (vermelho), `ghost` (transparente)\n- **Tamanhos**: `sm`, `md`, `lg`\n- **Props**: `fullWidth`, `disabled`, `className`\n\n#### Input (`components/ui/Input.tsx`)\n```tsx\n<Input\n  label=\"Email\"\n  type=\"email\"\n  error=\"Mensagem de erro\"\n  placeholder=\"seu@email.com\"\n  required\n/>\n```\n- Label autom\u00e1tico\n- Feedback de erro visual\n- Focus states\n- Tipos: text, email, password, number, date\n\n#### Card (`components/ui/Card.tsx`)\n```tsx\n<Card className=\"hover:shadow-md\">\n  <CardHeader>\n    <CardTitle>T\u00edtulo</CardTitle>\n  </CardHeader>\n  <CardContent>\n    Conte\u00fado\n  </CardContent>\n</Card>\n```\n- Bordas arredondadas\n- Sombra no hover\n- Padding padronizado\n\n#### Alert (`components/ui/Alert.tsx`)\n```tsx\n<Alert type=\"success\" title=\"Sucesso!\">\n  Opera\u00e7\u00e3o realizada com sucesso\n</Alert>\n```\n- **Tipos**: `success` (verde), `error` (vermelho)\n- T\u00edtulo opcional\n- \u00cdcone autom\u00e1tico\n\n#### Loading (`components/ui/Loading.tsx`)\n```tsx\n<Loading size=\"lg\" />\n```\n- Spinner com Loader2 (Lucide)\n- Centralizado\n- Tamanhos: sm, md, lg\n\n## \ud83d\udd10 Controle de Acesso\n\n- **P\u00e1ginas protegidas** com redirecionamento autom\u00e1tico para `/login`\n- Verifica\u00e7\u00e3o de autentica\u00e7\u00e3o via `useAuth()` hook\n- P\u00e1ginas exclusivas para admin:\n  - `/admin/users`\n- Valida\u00e7\u00f5es de permiss\u00e3o nas APIs\n\n## \ud83d\udcca Estados da Aplica\u00e7\u00e3o\n\n### Loading\n- Spinners durante requisi\u00e7\u00f5es\n- Bot\u00f5es desabilitados com loading\n- Estados vazios com CTAs\n- Mensagem \"Carregando...\"\n\n### Erros\n- Mensagens contextuais claras\n- Cores de destaque (vermelho)\n- Instru\u00e7\u00f5es de como resolver\n- Console.error para debug\n\n### Sucesso\n- Feedback positivo instant\u00e2neo\n- Redirecionamentos autom\u00e1ticos\n- Mensagens tempor\u00e1rias\n- Atualiza\u00e7\u00e3o de dados\n\n### Vazio\n- Mensagens amig\u00e1veis\n- \u00cdcones ilustrativos\n- Bot\u00f5es de a\u00e7\u00e3o (CTA)\n- Orienta\u00e7\u00e3o clara\n\n## \ud83c\udfae Fluxo de Uso\n\n### Usu\u00e1rio Comum\n1. Login com email/senha\n2. Visualizar dashboard com estat\u00edsticas pessoais\n3. Ver ranking geral\n4. Ver partidas existentes (com filtros)\n5. Criar nova partida (buscar jogadores, formar times)\n6. Adicionar rodadas durante o jogo\n7. Ver anima\u00e7\u00e3o de Lisa ao vencer com 0 pontos\n8. Alterar nome e senha em configura\u00e7\u00f5es\n\n### Administrador\n1. Todos os recursos do usu\u00e1rio comum\n2. Criar novos usu\u00e1rios\n3. Gerenciar lista de usu\u00e1rios\n4. Redefinir senhas de outros usu\u00e1rios\n5. Acesso completo ao sistema\n\n## \ud83c\udf10 API Integration\n\nTodas as p\u00e1ginas est\u00e3o integradas com as APIs:\n\n### Autentica\u00e7\u00e3o\n- `POST /api/auth/login` - Login\n- `POST /api/auth/logout` - Logout\n- `GET /api/auth/me` - Dados do usu\u00e1rio\n- `POST /api/auth/change-password` - Trocar senha\n- `POST /api/auth/update-profile` - Atualizar nome **NOVO!**\n\n### Admin\n- `GET /api/admin/users` - Listar usu\u00e1rios\n- `POST /api/admin/users` - Criar usu\u00e1rio\n- `PUT /api/admin/users/:id/password` - Redefinir senha\n\n### Partidas\n- `GET /api/games` - Listar partidas\n- `POST /api/games` - Criar partida (valida: sem m\u00faltiplas ativas)\n- `GET /api/games/:id` - Detalhes da partida\n- `GET /api/games/:id/rounds` - Listar rodadas\n- `POST /api/games/:id/rounds` - Adicionar rodada (valida: apenas membros)\n\n### Estat\u00edsticas **NOVO!**\n- `GET /api/stats/me` - Estat\u00edsticas do usu\u00e1rio logado\n- `GET /api/stats/ranking` - Ranking geral\n\n## \ud83d\udca1 Tecnologias Utilizadas\n\n- **Next.js 13.0.0**: Framework React com SSR\n- **TypeScript 4.9.4**: Tipagem est\u00e1tica\n- **Tailwind CSS 3.4.0**: Estiliza\u00e7\u00e3o utility-first\n- **Lucide React 0.562.0**: \u00cdcones\n- **clsx + tailwind-merge**: Utilit\u00e1rios CSS condicionais\n- **Firebase Admin SDK 11.0.0**: Firestore\n- **Next/Image**: Otimiza\u00e7\u00e3o de imagens (unoptimized)\n\n## \ud83d\ude80 Melhorias Implementadas\n\n- \u2705 Busca e filtros avan\u00e7ados\n- \u2705 Ranking com sistema de pontua\u00e7\u00e3o\n- \u2705 Estat\u00edsticas detalhadas\n- \u2705 Anima\u00e7\u00e3o de Lisa\n- \u2705 Edi\u00e7\u00e3o de perfil\n- \u2705 Nomes dos jogadores em vez de \"Time A/B\"\n- \u2705 Valida\u00e7\u00f5es de neg\u00f3cio (m\u00faltiplas partidas, apenas membros)\n- \u2705 L\u00f3gica correta do domin\u00f3 (100 pontos = perde)\n- \u2705 Loading states consistentes\n- \u2705 Responsividade completa\n\n## \ud83d\udcdd Pr\u00f3ximos Passos (Futuro)\n\n- [ ] Gr\u00e1ficos de desempenho ao longo do tempo\n- [ ] Sistema de conquistas/badges\n- [ ] Hist\u00f3rico de confrontos diretos\n- [ ] Modo escuro\n- [ ] PWA (Progressive Web App)\n- [ ] Notifica\u00e7\u00f5es push\n- [ ] Export de estat\u00edsticas (PDF/CSV)\n- [ ] Chat durante a partida\n- [ ] Modo torneio\n\n---\n\n**\ud83c\udfae Interface completa e otimizada para mobile-first!**\n
